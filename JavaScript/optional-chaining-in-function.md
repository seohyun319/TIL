`?.()` (optional chaning)
    
  함수 호출에서의 옵셔널 체이닝.

  Cannot invoke an object which is possibly 'undefined'. 일 때 사용 가능

  object에만 적용되는 문법인 줄 알았는데 다른 곳에도 적용 가능하다!

  ```jsx
  obj?.prop
  obj?.[expr]
  arr?.[index]
  func?.(args)
  ```

  [mdn](https://developer.mozilla.org/ko/docs/Web/JavaScript/Reference/Operators/Optional_chaining)
